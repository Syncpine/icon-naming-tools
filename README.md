# icon-naming-tools

这是一个更改文件名的工具

## Usage

双击打开程序

![](./img/ui_start.png "ui_start.png")

点击 【Open】, 选择源文件

![](./img/ui_btn_open_click.png "ui_btn_open_click.png")

点击 【Transform】, 更改文件名

![](./img/ui_btn_transform_click.png "ui_btn_transform_click.png")
